**Модуль:** ПМ.02 / ПМ.04 (Производственная практика)

##  Обзор проекта
Данный репозиторий содержит материалы по проектированию и разработке мобильного приложения для сферы электронной коммерции. Целью проекта является автоматизация процесса взаимодействия покупателя с торговой площадкой через мобильный интерфейс.

## 🛠 Технический стек
* **Язык разработки:** Python (Kivy/BeeWare) или Kotlin/Swift (в зависимости от целевой платформы).
* **Архитектура:** Clean Architecture (MVP/MVVM).
* **Передача данных:** REST API (JSON).
* **База данных (local):** SQLite для кеширования.

##  Реализованный функционал
В соответствии с техническим заданием реализованы следующие модули:
1. **Каталог товаров:** Динамическая загрузка данных и фильтрация.
2. **Корзина и заказы:** Логика добавления товаров и расчета итоговой стоимости.
3. **Профиль пользователя:** Авторизация и история транзакций.
4. **Интеграция с API:** Синхронизация остатков в реальном времени.

##  Тестирование
Проект прошел комплексную верификацию:
* Проверка корректности обработки API-запросов.
* Тестирование UI-компонентов на соответствие UX-гайдлайнам.
* Валидация безопасности платежных данных.
